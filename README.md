# Puppet Manifest Code 
## Purpose
 This repo helps define:
- What users should be on a server 
- What packages should be installed on a LAMP Stack Serve
## Files
- server_users_groups.pp - manages server users and groups 
- lamp_stack_server.pp - Installs packages needed for LAMP and a phpinfo.php page
- testing_puppet.pp - used for testing Puppet
